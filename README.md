# Codshop Landing Page

این پروژه شامل یک صفحه فرود مدرن برای معرفی و فروش محصولات فروشگاه آنلاین «کدشاپ» است که با الهام از طراحی permio.ir پیاده‌سازی شده است.

## پیش‌نمایش

برای مشاهده نسخه واقعی، کافی است فایل `index.html` را در مرورگر باز کنید.

## ساختار پروژه

- `index.html`: ساختار اصلی صفحه شامل بخش‌های مختلف مانند هدر، معرفی، محصولات، نظرات مشتریان و سوالات متداول.
- `styles.css`: استایل‌های کامل صفحه با طراحی واکنش‌گرا و استفاده از فونت فارسی Vazirmatn.
- `script.js`: اسکریپت‌های ساده برای کنترل منوی موبایل، تب‌های محصولات، اسلایدر نظرات و دکمه بازگشت به بالا.

## نحوه اجرا

1. مخزن را کلون کنید یا فایل‌ها را دانلود نمایید.
2. فایل `index.html` را با یک مرورگر مدرن (Chrome, Edge, Firefox) باز کنید.
3. برای اعمال تغییرات، ویرایشگر دلخواه خود را باز کرده و فایل‌های HTML/CSS/JS را ویرایش کنید.

## سفارشی‌سازی سریع

- برای تغییر تصاویر، آدرس آن‌ها را در تگ‌های `<img>` و یا در CSS جایگزین کنید.
- جهت ویرایش متن‌ها، کافی است متن فارسی موجود در HTML را تغییر دهید.
- رنگ‌های اصلی صفحه در ابتدای فایل `styles.css` و در متغیرهای CSS (`:root`) تعریف شده‌اند.

## لایسنس

این پروژه صرفاً برای نمایش و توسعه یک صفحه فرود نمونه ارائه شده است و استفاده تجاری آن منوط به رعایت حقوق تصاویر و فونت‌های استفاده شده می‌باشد.
